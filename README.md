# Turbo Grocery

Turbo Grocery is a web application for online grocery shopping.  
Users can browse products, search items, add them to cart, place orders, and manage their account.

## Features
- Product listing with search
- Add to Cart & Buy Now
- Order management (Ordered, Shipped, Delivered)
- Category-wise products
- User Signup, Login & Logout

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB
